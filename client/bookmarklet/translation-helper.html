Mozilla Translation Helper

by Mozilla Nepal

Use the following link (Pootle Translation Helper) to create a Bookmark for using Google Translate Power on Pootle (specifically to translate Firefox to Nepali on mozilla.locamotion.org). (Right Click, Bookmark this Link on Firefox) or (Drag to Bookmarks Bar, Ctrl + Shift + B to spawn the bar on Chrom(ium))

Use the following link (Launchpad Translation Helper) to create a Bookmark for using Google Translate Power on Launchpad (specifically to translate Ubuntu to Nepali on translations.launchpad.net ) (Right Click, Bookmark this Link on Firefox) or (Drag to Bookmarks Bar, Ctrl + Shift + B to spawn the bar on Chrom(ium))

How to Work with this?

After you bookmar the links, (and probably name it properly), you should visit either mozilla.locamotion.org or translations.launchpad.net's Nepali localization page. After you reach to localization page, (the one with text boxes) Please click the respective bookmarks. Once you click the empty input or textarea boxes you will find the translated text automatically (might take time, depends on your internet speed).

Special Instruction For launchpad translators

Since launchpad requires https, and browsers do not allow mixed contents to be loaded you should visit https://mozilla-nepal.org first and then Permanently add a security exception. Repeat the same for https://www.mozilla-nepal.org/translation-helper-pootle/translate-launchpad.js If you are stuck on how to do this, contact authors.
